# iniCommentProcessor
ini文件注释器

- 为ini文件一键在匹配关键字添加注释

高级用法：
需要更改config.ini

[Paths]
input_path = 
output_path = 
keywords_path = 

[function]
add = True
mode = 0
algorithm = 0 ;这个是算法，默认0为自动选择，1为算法1  2为算法2，建议无特殊需求不要更改

ps.
算法1可能没有算法2准确，如果遇到bug请选择算法2
